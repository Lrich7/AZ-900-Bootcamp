# Cloud Concepts

## What is Cloud Computing?

Cloud computing is the delivery of computing resources such as servers, storage, databases, networking, and software over the Internet instead of maintaining physical hardware.

## Benefits

- High availability
- Scalability
- Elasticity
- Reliability
- Cost efficiency

## Cloud Service Models

### IaaS

Infrastructure as a Service provides virtual machines, networking, and storage while the customer manages the operating system and applications.

### PaaS

Platform as a Service provides a managed platform for developing and deploying applications.

### SaaS

Software as a Service delivers complete applications over the Internet.

## My Notes

- Azure Virtual Machines are an example of IaaS.
- Azure App Service is an example of PaaS.
- Microsoft 365 is an example of SaaS.
